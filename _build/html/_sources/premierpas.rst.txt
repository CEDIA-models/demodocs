Installation de Compas
======================

On peut installer facilement Compas en utilisant pip ::

  pip install compas


Importer Compas dans un notebook ou script
==========================================

Pour importer compas dans un notebook ou un script python on ajoute ::

  import compas 

On peut aussi importer des sous-modules spécifiques en utilisant ::

  from compas import actors, demographics, data
